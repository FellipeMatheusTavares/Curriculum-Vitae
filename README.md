# Currículo Online — Fellipe Matheus Tavares Silva

## Sobre

Este projeto consiste no meu currículo online desenvolvido em HTML e hospedado através do GitHub Pages.

O objetivo é apresentar minha trajetória profissional, formação acadêmica, habilidades técnicas e experiências na área de Dados de forma acessível e profissional.

## Tecnologias usadas

- HTML5 & CSS3 puro (sem frameworks)
- Google Fonts (Inter + JetBrains Mono)
- Animações com Intersection Observer API
- Hospedagem via GitHub Pages
- Claude.ai para Desing
